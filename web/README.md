# 股票实时行情Web页面

一个简洁美观的股票实时行情展示页面，自动显示昨日成交量前十名的股票对应的当前实时行情数据。

## 功能特性

- 📊 **实时行情展示**: 自动获取并展示热门股票的实时行情数据
- 🔄 **自动刷新**: 每10秒自动刷新一次数据，保持数据实时性
- 📱 **响应式设计**: 适配桌面端和移动端设备
- 🎨 **美观界面**: 采用现代化深色主题设计，支持颜色区分涨跌
- 📈 **数据统计**: 显示上涨/下跌家数、平均涨跌幅、总成交额等统计信息
- 🔃 **手动刷新**: 支持手动刷新按钮，随时获取最新数据
- 📋 **表格排序**: 支持点击表头按不同字段排序

## 页面预览

页面包含以下主要区域：
1. **头部区域**: 显示标题、更新时间和刷新控制
2. **统计卡片**: 展示上涨家数、下跌家数、平均涨跌幅、总成交额
3. **行情表格**: 展示股票代码、名称、最新价、涨跌幅、涨跌额、成交量、成交额

## 使用方法

### 1. 安装依赖

```bash
cd web
pip install flask flask-cors
```

### 2. 启动服务

```bash
python app.py
```

### 3. 访问页面

打开浏览器访问: http://127.0.0.1:8080

## API接口

### 获取TOP10股票行情

- **URL**: `/api/top10`
- **Method**: GET
- **Response**:
```json
{
  "success": true,
  "data": [
    {
      "stock_code": "000001",
      "stock_name": "平安银行",
      "price": 10.50,
      "change": 0.25,
      "change_pct": 2.44,
      "volume": 1500000,
      "amount": 15750000
    }
  ],
  "update_time": "2025-02-03 18:12:00"
}
```

### 刷新数据

- **URL**: `/api/refresh`
- **Method**: GET
- **Response**: 同上

## 技术栈

- **后端**: Flask + Flask-CORS
- **前端**: HTML5 + CSS3 + JavaScript (原生)
- **数据源**: 新浪财经API (通过adata库)

## 注意事项

1. 当前版本使用预设的热门股票列表进行演示
2. 自动刷新默认开启，每10秒刷新一次
3. 页面采用深色主题，涨跌用颜色区分（红色跌/绿色涨）
4. 支持响应式布局，适配移动端访问
