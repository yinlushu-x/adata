# AData 股票K线图绘制功能使用示例

## 概述
AData新增了股票K线图绘制功能，允许用户通过简单的接口获取股票行情数据并绘制K线图。

## 安装
```bash
pip install adata
```

## 基本使用

### 1. 绘制股票K线图
```python
import adata

# 绘制股票000001的日K线图（2024年全年）
adata.chart.draw_kline(
    stock_code='000001',
    start_date='2024-01-01',
    end_date='2024-12-31'
)
```

### 2. 自定义参数
```python
import adata

# 绘制股票000001的周K线图（2024年全年，后复权）
adata.chart.draw_kline(
    stock_code='000001',
    start_date='2024-01-01',
    end_date='2024-12-31',
    k_type=2,  # 周K线
    adjust_type=2  # 后复权
)
```

## 参数说明
- `stock_code`: 股票代码（如：000001）
- `start_date`: 开始日期（格式：YYYY-MM-DD）
- `end_date`: 结束日期（格式：YYYY-MM-DD）
- `k_type`: K线类型（可选值：1.日；2.周；3.月；4.季度；5.5分钟；15.15分钟；30.30分钟；60.60分钟），默认值：1（日K线）
- `adjust_type`: 复权类型（可选值：0.不复权；1.前复权；2.后复权），默认值：1（前复权）

## 输出结果
- 成功绘制K线图并显示
- 图表包含：
  - 价格K线图（OHLC）
  - 成交量柱状图
  - 时间轴
  - 价格标签
  - 标题和图例

## 注意事项
1. 需要安装依赖库：`mplfinance`（已包含在AData安装中）
2. 网络请求可能需要代理，可使用`adata.proxy()`设置代理
3. 部分股票可能没有完整的历史数据
4. 图表显示需要支持图形界面的环境

## 更多功能
- 查看其他数据：`adata.stock.info.all_code()`获取所有股票代码
- 获取行情数据：`adata.stock.market.get_market()`
- 代理设置：`adata.proxy(is_proxy=True, ip='代理地址')`
