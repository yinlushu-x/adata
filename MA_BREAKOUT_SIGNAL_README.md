# 均线突破 + 成交量确认信号检测模块

## 功能介绍

该模块实现了一个“均线突破 + 成交量确认”的信号检测功能，用于识别股票的买入信号。

## 功能特点

1. **数据获取**：使用adata.stock.market.get_market(...)获取指定股票的日线行情数据
2. **技术指标计算**：计算MA5、MA10、MA20均线和成交量10日均值
3. **信号检测**：根据严格的规则检测均线突破和成交量确认信号
4. **信号输出**：以表格形式输出信号日期、收盘价和成交量
5. **可视化**：绘制K线或收盘价折线图，叠加均线，并在信号点位置绘制标记

## 使用方法

```python
from adata.stock.market import detect_ma_breakout_signal

# 测试示例：平安银行(000001)
detect_ma_breakout_signal('000001', '2024-01-01', '2024-06-30')
```

## 信号定义

### 均线突破条件：
- 当日 close > MA20
- 前一日 close <= MA20

### 成交量确认条件：
- 当日 volume > VOL_MA10

### 有效买入信号：
同时满足均线突破和成交量确认条件

## 可视化

模块会自动绘制包含以下元素的图表：
- 收盘价折线图
- MA5、MA10、MA20均线
- 信号点（红色向上箭头）

## 注意事项

1. 模块使用matplotlib进行可视化，需要确保matplotlib库已安装
2. 模块支持中文显示，已设置中文字体
3. 若区间内未检测到有效信号，会明确提示"区间内未检测到有效信号"
4. 若未获取到有效行情数据，会提示"未获取到有效行情数据"
